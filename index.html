DRAFTKINGS NBA CHEAT SHEET

How to Use the DraftKings NBA Cheat Sheet
1. Cheat Sheet View
  • • Switch positions using the tabs at the top
  • • Click any column header to sort (Rank, Value, Points, etc.)
  • • Use the filters above to narrow by slate, team, salary, etc.
  • • Green Value = Excellent (7x or higher)
    Red Value = Avoid (4x or lower)
2. Player Details
  • • Click any player name to open their detailed modal
  • • View recent game log, projections, news & injury updates, matchup breakdown
  • • Use this info to make smarter lineup decisions
3. Lineup Builder
  • • Click "Switch to Lineup View"
  • • Browse players by position and click the + button to add them
  • • Build a full 8-player DraftKings lineup (respects position rules & $50,000 salary cap)
  • • Remove players with the – button
4. Pro Tips
  • • Prioritize high-value (green) players
  • • Check the Game Log tab for recent form
  • • Use the Matchup tab to exploit favorable defenses
  • • Export your finished lineup with one click
